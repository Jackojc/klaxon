# Klaxon

A minimalist RPN language that compiles to native code.

Goals:
- Minimal
- Simple Implementation
- No Dependencies
- Cross Platform
- Composable & Modular

Features:
- Register Allocation
- Procedures
- x86-64 Code Generation
- Optimisation Passes
	- Inlining
	- Dead Code Elimination
	- Constant Folding
	- Strength Reduction
	- Instruction Scheduling
	- Superoptimisation

Semantics:
- Word Size Values
- UTF-8 Source Encoding
- Two's Complement Signed Arithmetic
